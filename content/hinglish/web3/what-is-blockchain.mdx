# What is Blockchain?

Ek **blockchain** ek distributed database hota hai jo continuously grow hone wali ordered records ki list maintain karta hai, jinhe *blocks* kehte hain. Yeh blocks cryptography ka use karke aapas mein linked hote hain. Har block mein pichle block ka cryptographic hash, ek timestamp, aur transaction data hota hai.

## Key Concepts (Zaruri Baten)

- **Decentralization**: Kisi ek central server ke bajaye, ledger network ke sab computers mein faila hua hota hai.
- **Immutability**: Ek baar data record ho gaya toh usay badalna lagbhag na-mumkin hota hai.
- **Consensus Mechanisms**: Proof of Work (PoW) ya Proof of Stake (PoS) jaise systems yeh make sure karte hain ki sabhi nodes ledger ke state par agree karein.
